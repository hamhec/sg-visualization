info About

DAMN is a defeasible reasoning platform available on the web. It allows the use of different defeasible reasoning semantics (ambiguity blocking/propagating with or without team defeat) in order to reason with incoherent or inconsistent knowledge of different agents.

It integrates a defeasible reasoning tool with agent collaboration and visualization features. It is geared towards decision making where each agent has its own knowledge base that can be combined with other agents to detect and visualize conflicts and potentially solve them using a semantics.

Click on Help (top right corner) for more information on how to use the platform, or browse the default examples below.

Source code is available at Github page.
